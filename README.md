# E.NT
dark mode for ENT

Ever thought the ENT was too bright? Well now it's dark like discord's dark mode.
It has customisation features such as changing colors, adding radii to borders... thats it for now.

If you're using this on firefox, just be aware that while testing on linux, kde plasma takes over the color picker, so you have to imput the actual hex color codes into the text boxes. yes the codes are text boxes, you can edit them. they will automatically add # if you didnt do it already
